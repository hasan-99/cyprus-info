# Cyprus Company Registration Landing Page

## Overview
Bilingual (Arabic + English) landing page for a Cyprus company registration service. Targeting Arabic-speaking entrepreneurs, investors, and business owners who want to register a European company in Cyprus for EUR 999 + VAT.

## Tech Stack
- **Frontend:** Plain HTML, CSS, JavaScript (no framework)
- **Styling:** Custom CSS with RTL/LTR support
- **Deployment:** Vercel (static)
- **Form:** Static (no backend for now)
- **i18n:** Arabic (RTL) + English (LTR) with language switcher

## Business Context
- **Service:** Cyprus EU company registration
- **Price:** EUR 999 + VAT
- **Timeline:** ~2 weeks after document submission
- **Includes:** 4 months free secretarial + registered address
- **Key USP:** Fully remote, no travel needed
- **Target:** Arab investors, entrepreneurs, tech companies, international traders
- **Tax angle:** 12.5% corporate tax, IP Box Regime (2.5% effective for qualifying IP)

## Brand Direction
- Design from scratch — premium, trustworthy, professional
- Must feel European/corporate but appeal to Arabic-speaking audience
- Bilingual: Arabic RTL as primary, English LTR as secondary
- No generic blue-purple gradients, no card grids
- Distinctive fonts, sharp accents, layered backgrounds, subtle gradients

## Constraints
- No backend/database
- No framework — plain HTML/CSS/JS (GSAP via CDN approved)
- Must be fully responsive (mobile-first)
- Bilingual with smooth language toggle
- Form is static for now (no submission handler)
- Deploy to Vercel as static site
